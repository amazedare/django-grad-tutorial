# Step 2: Create Django Project

## Instructions

### 1. Create Django Project

```bash
django-admin startproject blogs .
```

### 2. Verify Project Structure

```bash
ls -la
```

### 3. Test the Project

```bash
python manage.py runserver
```

Stop the server with `Ctrl+C`.
